The frequency list is the file to import directly into Wireless Workbench (Import > frequencies from file). The reference sheet maps each frequency to a suggested name and its coordination group — Ofcom licences don't include per-mic names, so use it to manually label channels in WWB.

Download WWB7 show file (.shw)
The .shw show file is experimental. Shure doesn't document WWB's native show-file format, so this was built by reverse-engineering a real working file — it has not been validated by Shure. Receiver IP addresses in particular are a best-effort guess (we've never seen a real show file with one configured) — verify/set them again inside WWB if they don't take. Open the file in Wireless Workbench and check it carefully before relying on it for a real show.
